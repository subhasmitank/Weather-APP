# 🌦️ Basic Weather App

A simple and responsive weather application built using HTML, CSS, and JavaScript. It fetches real-time weather information using the OpenWeather API.

## 🚀 Features

- Search weather by city name
- Displays temperature, humidity, and weather condition
- Responsive design
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API
